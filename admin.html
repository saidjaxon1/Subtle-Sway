Connect to GitHub

This panel saves your changes straight to the GitHub repository, so it needs a personal access token. The token is stored only in this browser — never on any server.

  1. On GitHub go to Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  2. Repository access: Only select repositories → choose Subtle-Sway.
  3. Permissions: Contents → Read and write. Generate and copy the token.
  4. Paste it below and press Connect.
